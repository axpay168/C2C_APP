# 後端安裝說明

## 環境要求

- PHP >= 7.2（建議 7.4）
- 擴展：ext-json, ext-curl, ext-pdo, ext-bcmath, **ext-fileinfo**
- Composer

## 安裝步驟

```bash
cd /www/wwwroot/old/backend-php
composer install
```

若報錯缺少 **ext-fileinfo**：在 php.ini 中啟用（取消註釋或添加）：

```ini
extension=fileinfo
```

若本機無法執行 composer（如權限、擴展不全），可在本機或另一台環境完整的機器上執行 `composer install`，再將整個 `vendor` 目錄上傳到伺服器。

## 數據庫

1. 創建數據庫（如 `hkex_old`），utf8mb4
2. 導入：`mysql -u root -p hkex_old < ../db/schema_full.sql`
3. 修改 `application/database.php` 中的 database、username、password

## 運行

Nginx 站點根目錄指向 `public/`，或使用 PHP 內建服務：  
`php think run`（需在專案根目錄，且 ThinkPHP 支援時）。
