# 本地 8082 啟動說明

## 已修復

1. **prestart 腳本**：`npm run start` 前會自動釋放 8082 端口
2. **服務目錄**：使用 `static-h5`（根路徑直接顯示 H5）
3. **xsel 錯誤**：已加上 `-n` 參數抑制

## 使用方式

```bash
cd /www/wwwroot/old/frontend-uniapp
npm run start
```

訪問：**http://localhost:8082**

若仍提示端口被佔用，可先手動釋放：

```bash
fuser -k 8082/tcp
sleep 2
npm run start
```

## MCP 瀏覽器測試說明

MCP 瀏覽器可能無法直接訪問 `localhost:8082`（不同網路環境）。  
生產環境 **https://mxtrx.top/h5/** 可正常使用，可於該處進行 UI 測試。
