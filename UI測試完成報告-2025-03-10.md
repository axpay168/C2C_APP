> 測試日期：2025-03-10

## 一、已修復問題

### 1. 註冊 API：mkdir Permission denied
- **原因**：`application/config.php` 中 cache path 指向 `APP_PATH . 'runtime/lang_cache/'`，該目錄不存在且 PHP (www 用戶) 無權建立
- **修復**：將 cache path 改為 `CACHE_PATH`（即 `runtime/cache/`），並已建立 `runtime` 目錄並設定權限

### 2. Auth.php 參數順序（PHP 8 相容）
- **原因**：`register($username, $password, $extend = [], $invid, $lang)` 中可選參數在必填參數前
- **修復**：改為 `register($username, $password, $invid, $lang, $extend = [])` 並更新呼叫處

### 3. runtime 目錄與權限
- 已建立 `/www/wwwroot/old/backend-php/runtime/{cache,log,temp,template,schema,chunks}`
- 已設定 `chown www:www` 及 `chmod 777`

## 五、H5 最佳實踐參考

1. **表單提交**：建議使用 `<form @submit>` 搭配 `formType="submit"`，避免 view 點擊觸發
2. **按鈕**：按鈕不宜包成子組件再使用 `formType`，否則可能無法觸發提交
3. **密碼欄位**：避免「Password field is not contained in a form」警告，可考慮使用 `<form>` 包住輸入區
4. **meta**：`apple-mobile-web-app-capable` 已棄用，可改為 `mobile-web-app-capable`
