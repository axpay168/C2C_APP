# 🎯 超簡單測試清單

## 📋 使用說明

1. 打開 HBuilderX 預覽（應該已經打開了）
2. 打開瀏覽器開發者工具（F12）
3. 按照下面的清單逐一測試
4. 每個測試完成後，標記 ✅ 或 ❌
5. 如果有問題，告訴我錯誤訊息，我立即修復

---

## 🧪 測試 1：註冊功能

### 1.1 郵箱為空 ❌/✅
- [ ] 不輸入任何內容，點擊「註冊」
- [ ] 是否顯示「请输入邮箱」？
- [ ] 如果有問題，告訴我：_________________

### 1.2 密碼為空 ❌/✅
- [ ] 只輸入郵箱，不輸入密碼，點擊「註冊」
- [ ] 是否顯示「密码不能低于6位」？
- [ ] 如果有問題，告訴我：_________________

### 1.3 郵箱格式錯誤 ❌/✅
- [ ] 輸入 `test@`，點擊「註冊」
- [ ] 是否顯示「请输入正确的邮箱格式」？
- [ ] 如果有問題，告訴我：_________________

### 1.4 密碼長度不足 ❌/✅
- [ ] 輸入密碼 `12345`（5位），點擊「註冊」
- [ ] 是否顯示「密码不能低于6位」？
- [ ] 如果有問題，告訴我：_________________

### 1.5 兩次密碼不一致 ❌/✅
- [ ] 密碼：`123456`，確認密碼：`1234567`，點擊「註冊」
- [ ] 是否顯示「两次密码不一致」？
- [ ] 如果有問題，告訴我：_________________

### 1.6 成功註冊 ⭐ 重要 ❌/✅
- [ ] 輸入：郵箱 `test${Date.now()}@example.com`，密碼 `123456`，確認密碼 `123456`
- [ ] 點擊「註冊」
- [ ] 是否顯示「注册成功」？
- [ ] 是否跳轉到登入頁面？
- [ ] **請記錄註冊的帳號**：`_________________`
- [ ] **請記錄註冊的密碼**：`_________________`
- [ ] 如果有問題，告訴我：_________________
