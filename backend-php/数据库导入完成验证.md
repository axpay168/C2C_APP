## 🐛 如果验证失败

### 问题1：数据库连接失败

**可能原因**：
- MySQL服务未运行
- 数据库未创建
- 数据库用户或密码错误

**解决方法**：
1. 在宝塔面板中检查MySQL服务是否运行
2. 确认数据库 `hkex02` 已创建
3. 确认数据库用户 `hkex02` 存在且有权限
4. 检查 `application/database.php` 配置文件

### 问题2：表数量不对

**可能原因**：
- SQL文件未完全导入
- 导入过程中出现错误

**解决方法**：
1. 检查导入日志是否有错误
2. 重新导入SQL文件
3. 在phpMyAdmin中手动检查表

### 问题3：管理员账号不存在

**可能原因**：
- SQL文件未完全导入
- 管理员数据未导入

**解决方法**：
1. 检查 `fa_admin` 表是否有数据
2. 如果没有数据，可能需要手动创建管理员账号
3. 或重新导入SQL文件

### 问题4：后台无法访问

**可能原因**：
- Nginx配置问题
- PHP-FPM未运行
- 文件权限问题

**解决方法**：
1. 检查Nginx服务是否运行
2. 检查PHP-FPM服务是否运行
3. 检查网站根目录配置是否正确
4. 参考：`修复404错误指南.md`
