### 检查表数量
```sql
SELECT COUNT(*) as table_count 
FROM information_schema.tables 
WHERE table_schema = 'hkex02' 
AND table_name LIKE 'fa_%';
```

### 检查管理员账号
```sql

FROM fa_admin
```

```sql
SELECT name, value, type 
FROM fa_config 
WHERE name IN ('site_name', 'site_beian', 'version');
```

## ⚠️ 注意事项

1. **备份数据**：重建数据库前，如果有重要数据请先备份

3. **权限**：确保数据库用户有足够的权限
4. **导入顺序**：建议按顺序导入两个SQL文件
