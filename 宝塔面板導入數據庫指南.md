# old 專案 - 宝塔面板導入數據庫指南

## 一、創建數據庫

1. 登錄宝塔面板 → **數據庫**
2. 點擊 **添加數據庫**
3. 數據庫名：`hkex_old`
4. 用戶名：可選新建或用現有（如 `root` 或 `hkex02`）
5. 字符集：`utf8mb4`
6. 提交

## 二、導入 SQL

### 方法 A：宝塔數據庫管理

1. 找到數據庫 **`hkex_old`**
2. 點擊右側 **導入**
3. 選擇文件：`/www/wwwroot/old/db/schema_full.sql`
4. 點擊 **開始導入**
5. 等待完成

### 方法 B：phpMyAdmin

1. 點擊數據庫 `hkex_old` 的 **管理** / **phpMyAdmin**
2. 左側選擇 `hkex_old`
3. 頂部 **導入** → 選擇 `schema_full.sql`
4. 執行

## 三、修改後端配置

編輯 `/www/wwwroot/old/backend-php/application/database.php`：

- `database` 改為：`hkex_old`
- `username` / `password` 改為宝塔中該數據庫的用戶和密碼

## 四、驗證

- 后台：`http://您的域名/old后台路徑/index/login`
- 默認：admin / 123456
