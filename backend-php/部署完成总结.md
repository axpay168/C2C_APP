### 2. Web服务器配置

#### Nginx配置示例
创建或编辑 `/etc/nginx/sites-available/MXTRX.TOP`:

```nginx
server {
    listen 80;
    server_name MXTRX.TOP;
    root /www/wwwroot/hkex-sm.com/public;
    index index.php index.html;

    access_log /var/log/nginx/MXTRX.TOP.access.log;
    error_log /var/log/nginx/MXTRX.TOP.error.log;

    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }

    location ~ \.php$ {
        fastcgi_pass unix:/var/run/php-fpm/php-fpm.sock;
        fastcgi_index index.php;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        include fastcgi_params;
    }

    location ~ /\.ht {
        deny all;
    }
}
```

启用配置:
```bash
sudo ln -s /etc/nginx/sites-available/MXTRX.TOP /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

#### Apache配置
确保已启用mod_rewrite:
```bash
sudo a2enmod rewrite
sudo systemctl restart apache2
```
