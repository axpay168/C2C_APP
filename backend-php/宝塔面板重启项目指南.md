# 宝塔面板重启项目指南

## 🚀 快速重启步骤

### 方法一：通过软件商店重启（推荐）

#### 第一步：重启Nginx

1. **登录宝塔面板**
2. **点击左侧菜单"软件商店"**
3. **找到"Nginx"** → 点击右侧的 **"设置"** 按钮
4. **进入"服务"标签**
5. **点击"重启"按钮**
6. **等待重启完成**

#### 第二步：重启PHP-FPM

1. **在"软件商店"中找到"PHP 8.2"**（或您使用的PHP版本）
2. **点击右侧的"设置"按钮**
3. **进入"服务"标签**
4. **点击"重启"按钮**
5. **等待重启完成**

### 方法二：通过网站管理重启

1. **登录宝塔面板**
2. **点击左侧菜单"网站"**
3. **找到域名 `MXTRX.TOP` 的网站**
4. **点击右侧的"重启"或"重载"按钮**

### 方法三：一键重启所有服务

1. **登录宝塔面板**
2. **点击左侧菜单"软件商店"**
3. **依次重启以下服务**：
   - Nginx → 设置 → 服务 → 重启
   - PHP 8.2 → 设置 → 服务 → 重启
   - MySQL（如果需要）→ 设置 → 服务 → 重启

## 🔧 清理缓存（可选）

### 清理FastAdmin运行时缓存

1. **登录宝塔面板**
2. **点击左侧菜单"文件"**
3. **进入项目目录**：`/www/wwwroot/hkex-sm.com`
4. **删除以下目录中的缓存文件**：
   - `runtime/cache/` - 应用缓存
   - `runtime/temp/` - 临时文件
   - `runtime/log/` - 日志文件（可选）

或者使用SSH执行：
```bash
cd /www/wwwroot/hkex-sm.com
rm -rf runtime/cache/*
rm -rf runtime/temp/*
```

## ✅ 验证重启

重启完成后，测试以下URL：

1. **测试后台入口**：
   ```
   http://MXTRX.TOP/buSRxMqJOo.php
   ```

2. **测试登录页面**：


3. **测试API接口**：
   ```
   http://MXTRX.TOP/api/index/index
   ```

## 🐛 如果重启后仍有问题

### 检查服务状态

在宝塔面板中检查：
1. **软件商店** → **Nginx** → **设置** → **服务** → 查看状态
2. **软件商店** → **PHP 8.2** → **设置** → **服务** → 查看状态

### 查看错误日志

1. **网站** → **MXTRX.TOP** → **设置** → **日志**
2. 查看"错误日志"和"访问日志"

### 检查文件权限

确保以下目录有写入权限：
- `runtime/` - 755权限
- `public/uploads/` - 755权限

## 📋 重启检查清单

- [ ] Nginx服务已重启
- [ ] PHP-FPM服务已重启
- [ ] 缓存已清理（可选）
- [ ] 文件权限正确
- [ ] 网站可以正常访问
- [ ] 后台可以正常登录

## ⚡ 快速命令（SSH权限）

如果有SSH权限，可以使用：

```bash
cd /www/wwwroot/hkex-sm.com
./重启项目.sh
```

或者手动执行：

```bash
# 清理缓存
rm -rf runtime/cache/* runtime/temp/*

# 重启Nginx
systemctl restart nginx
# 或
nginx -s reload

# 重启PHP-FPM（根据版本）
systemctl restart php-fpm-82
# 或
systemctl restart php82-php-fpm
```

---

**提示**：如果使用宝塔面板，建议通过面板界面重启服务，这样更安全可靠。
