# HBuilderX 編譯錯誤快速修復指南

## 錯誤訊息
```
文件查找失敗：'echarts' at main.js:20
文件查找失敗：'ethers' at pages\index\index.vue:25
文件查找失敗：'tronweb' at pages\index\index.vue:524
```

## 快速解決（3 步驟）

### 步驟 1：執行修復腳本
雙擊執行：`修復HBuilderX依賴問題.bat`

### 步驟 2：等待安裝完成
腳本會自動安裝所需依賴（echarts、ethers、tronweb）

### 步驟 3：重新編譯
在 HBuilderX 中執行：**發行 → 網站-H5**

---

## 如果腳本無法執行

### 方法 A：在 HBuilderX 中安裝（最簡單）
1. 開啟 HBuilderX
2. 開啟專案目錄
3. 在 HBuilderX 底部終端執行：
   ```bash
   npm install
   ```
4. 重新編譯

### 方法 B：手動安裝 Node.js
1. 下載：https://nodejs.org/
2. 安裝時勾選「Add to PATH」
3. 重新開啟命令提示字元
4. 進入專案目錄執行：`npm install`

### 方法 C：使用已編譯版本
直接使用 `static-h5/` 目錄（無需編譯）
