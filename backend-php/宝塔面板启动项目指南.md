# 宝塔面板启动项目指南

## 🚀 快速启动步骤

### 第一步：启动MySQL服务

1. **登录宝塔面板**
2. **点击左侧菜单"软件商店"**
3. **找到"MySQL"** → 点击右侧的 **"设置"** 按钮
4. **进入"服务"标签**
5. **点击"启动"按钮**（如果未运行）
6. **等待启动完成**

### 第二步：启动PHP-FPM服务

1. **在"软件商店"中找到"PHP 8.2"**（或您使用的PHP版本）
2. **点击右侧的"设置"按钮**
3. **进入"服务"标签**
4. **点击"启动"按钮**（如果未运行）
5. **等待启动完成**

### 第三步：启动Nginx服务

1. **在"软件商店"中找到"Nginx"**
2. **点击右侧的"设置"按钮**
3. **进入"服务"标签**
4. **点击"启动"按钮**（如果未运行）
5. **等待启动完成**

### 第四步：验证服务状态

在宝塔面板中检查所有服务是否运行：
- ✅ MySQL：运行中
- ✅ PHP 8.2：运行中
- ✅ Nginx：运行中

## 🔧 修复文件权限

### 在宝塔面板中设置权限

1. **点击左侧菜单"文件"**
2. **进入项目目录**：`/www/wwwroot/hkex-sm.com`
3. **选中以下目录**，点击右键 → **"权限"**：
   - `runtime/` → 设置为 **755**
   - `public/uploads/` → 设置为 **755**

### 或使用SSH命令

```bash
cd /www/wwwroot/hkex-sm.com
chmod -R 755 runtime/
chmod -R 755 public/uploads/
```

## 🐛 常见问题解决

### 问题1：数据库连接失败

**原因**：MySQL服务未运行或数据库未创建

**解决**：
1. 在宝塔面板中启动MySQL服务
2. 检查数据库是否已创建
3. 检查数据库用户和密码是否正确
4. 参考：`数据库重建指南.md`

### 问题2：502 Bad Gateway

**原因**：PHP-FPM服务未运行

**解决**：
1. 在宝塔面板中启动PHP-FPM服务
2. 检查PHP版本是否正确
3. 检查Nginx配置中的PHP-FPM路径

### 问题3：403 Forbidden

**原因**：文件权限问题

**解决**：
1. 设置 `runtime/` 目录权限为 755
2. 设置 `public/uploads/` 目录权限为 755
3. 确保目录所有者正确（通常是 www）

### 问题4：404 Not Found

**原因**：网站根目录配置错误

**解决**：
1. 在宝塔面板中检查网站设置
2. 确保网站根目录为：`/www/wwwroot/hkex-sm.com/public`
3. 参考：`修复404错误指南.md`

## ⚡ 快速启动命令（SSH权限）

如果有SSH权限，可以使用：

```bash
cd /www/wwwroot/hkex-sm.com
./启动项目.sh
```

或者手动启动服务：

```bash
# 启动MySQL
systemctl start mysql
# 或
systemctl start mysqld

# 启动PHP-FPM
systemctl start php-fpm-82
# 或
systemctl start php82-php-fpm

# 启动Nginx
systemctl start nginx
```
