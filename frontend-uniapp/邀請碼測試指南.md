# 邀請碼測試指南

## 📋 如何獲取正確的邀請碼

### 方法 1：通過網頁查詢（推薦）

1. **訪問查詢頁面**：
   ```
   http://your-domain/index.php/query_invite_code.php
   ```
   或
   ```
   http://localhost/index.php/query_invite_code.php
   ```

2. **查看可用邀請碼**：
   - 頁面會顯示所有已註冊用戶的邀請碼
   - 推薦使用第一個用戶的邀請碼進行測試

---

### 方法 2：直接查詢數據庫

**SQL 查詢語句**：
```sql
SELECT id, username, email, code 
FROM fa_user 
WHERE status = 'normal' AND code != '' 
ORDER BY id ASC 
LIMIT 10;
```

**獲取第一個邀請碼**：
```sql
SELECT code FROM fa_user WHERE status = 'normal' AND code != '' ORDER BY id ASC LIMIT 1;
```

---

### 方法 3：使用默認測試邀請碼

如果數據庫中已有用戶（如默認的 admin 用戶），可以：

1. **檢查 admin 用戶是否有邀請碼**：
   ```sql
   SELECT code FROM fa_user WHERE username = 'admin';
   ```

2. **如果沒有，需要先註冊一個用戶**：
   - 註冊時會自動生成 6 位數字的邀請碼
   - 該邀請碼存儲在 `code` 字段中

---

## 🧪 測試步驟

### 步驟 1：獲取邀請碼

使用上述任一方法獲取一個有效的邀請碼（例如：`123456`）

### 步驟 2：測試註冊功能

1. **打開註冊頁面**
2. **輸入測試數據**：
   - 郵箱：`test@example.com`
   - 密碼：`123456`
   - 確認密碼：`123456`
   - **邀請碼：`[你獲取的邀請碼]`**
3. **點擊註冊**

### 步驟 3：驗證結果

**成功情況**：
- ✅ 顯示「注册成功」
- ✅ 跳轉到登入頁面
- ✅ 數據庫中新增用戶記錄
- ✅ 新用戶自動生成新的邀請碼

**失敗情況**：
- ❌ 邀請碼為空 → 顯示「请输入邀请码」
- ❌ 邀請碼不存在 → 後端返回錯誤（但當前代碼中此驗證被註釋了）

---

## 📝 邀請碼格式說明

根據 `Auth.php` 中的 `createCode()` 方法：

- **長度**：6 位
- **格式**：純數字（`NUMBER` 格式）
- **字符集**：`0123456789`
- **唯一性**：系統會確保每個邀請碼都是唯一的

---

## 🔍 常見問題

### Q1：為什麼查詢不到邀請碼？

**可能原因**：
1. 數據庫中還沒有註冊用戶
2. 用戶的 `code` 字段為空
3. 用戶狀態不是 `normal`

**解決方案**：
- 先註冊一個測試用戶
- 檢查數據庫中 `fa_user` 表的數據

### Q2：如何創建測試邀請碼？

**方法**：
1. 通過後台管理系統創建用戶
2. 或直接註冊一個用戶（需要先有一個邀請碼）
3. 或手動在數據庫中插入測試數據

**手動插入測試數據（僅用於開發測試）**：
```sql
-- 更新現有用戶的邀請碼
UPDATE fa_user SET code = '123456' WHERE id = 1;

-- 或插入測試用戶
INSERT INTO fa_user (username, password, code, status, createtime) 
VALUES ('testuser', '', '123456', 'normal', UNIX_TIMESTAMP());
```

## 🚀 快速測試

**最簡單的方法**：

1. 訪問 `http://your-domain/index.php/query_invite_code.php`
2. 複製頁面上顯示的第一個邀請碼
3. 在註冊頁面使用該邀請碼進行測試

---

## ⚠️ 注意事項

1. **邀請碼驗證**：
   - 前端已啟用邀請碼必填驗證
   - 後端要求邀請碼不能為空
   - 後端會檢查邀請碼是否存在（但當前代碼中此驗證被註釋了）

2. **測試環境**：
   - 確保數據庫連接正常
   - 確保後端 API 可以正常訪問

3. **生產環境**：
   - 不要使用測試邀請碼
   - 確保邀請碼的唯一性和安全性
