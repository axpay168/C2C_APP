# H5 前端建置指南

修改 `pages/common/login.vue` 等原始碼後，需要重新建置 H5 才能生效。建置輸出需覆蓋到 `public/h5/` 目錄。

## 方法一：使用 HBuilderX（建議）

若專案是用 HBuilderX 建立的：

1. 使用 **HBuilderX** 開啟專案目錄：`数据库前端/ouyuan 2/`
2. 選單：**發行** → **網站-H5 行動版**
3. 建置完成後，輸出在 `unpackage/dist/build/h5/`
4. 將該目錄內所有檔案複製到 `public/h5/`，覆蓋原檔案

```bash
# 在專案根目錄執行
cp -r "数据库前端/ouyuan 2/unpackage/dist/build/h5/"* public/h5/
```

## 方法二：使用 npm 命令列

### 前提條件
- 已安裝 Node.js 與 npm
- 專案已安裝 uni-app CLI 相關依賴

### 步驟

1. 安裝必要依賴（若尚未安裝）：
```bash
cd "数据库前端/ouyuan 2"
npm install
# 若無 uni-app CLI，可嘗試：
# npm install @dcloudio/uni-cli-vue2 sass sass-loader@10 cross-env --save-dev
```

2. 執行建置：
```bash
npm run build:h5
```

3. 複製建置輸出到 public/h5：
```bash
cp -r unpackage/dist/build/h5/* ../../public/h5/
```
