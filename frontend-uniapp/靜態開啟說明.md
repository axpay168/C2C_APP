# 前端靜態獨立開啟（不依賴 HBuilderX）

## 最快方式

```bash
cd /www/wwwroot/old/frontend-uniapp
npm install --legacy-peer-deps   # 僅首次
npm run start
```

瀏覽器打開：**http://localhost:8082**

- 使用目錄 `static-h5/` 內已編譯好的 H5，無需 HBuilderX，無需先編譯
- 適合先靜態測試、對接 API、完善介面

## 其他命令

- `npm run dev:h5` — 開發模式（改源碼即時預覽）
- `npm run build:h5` — 編譯 H5
- `npm run deploy:h5` — 將 H5 複製到後端 `public/h5/`

詳見專案根目錄 **運行和部署指南.md**。
