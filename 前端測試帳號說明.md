```bash

```

## 📝 登入方式說明

根據登入 API，前端登入可以使用以下方式之一：
1. **郵箱**：使用用戶的郵箱地址
2. **手機號**：使用用戶的手機號碼
3. **用戶名**：使用用戶名（如果支持）

## 🔍 查找密碼的方法

如果您需要查找或重置某個用戶的密碼：

1. **查看項目文檔**：檢查是否有密碼記錄文檔
2. **聯繫項目管理員**：詢問是否有測試帳號密碼
3. **重置密碼**：通過後台管理功能重置用戶密碼
4. **創建新帳號**：使用註冊功能創建新的測試帳號

## 💡 建議

**最佳測試方式**：
1. 使用前端註冊功能創建一個新的測試帳號
2. 記錄帳號和密碼
3. 使用新創建的帳號測試登入功能

這樣可以確保：
